Demonstrated use of REST using Git API

This page retrieves a git user and repository information.
Enter a git user name in the text box below, click the 'Get Gut Hub Detail' button, to see the details The Table on the left shows user information, while the one on the right shows repository details for that user
The technology used is HTML/CSS/Angular and REST using the $http library in conjunction with a promise-then logic

Pending and Refinements
While the page is functionally complete and sufficiently readable, some items have to be addressed; others can be improved:
  • The page has to be refreshed when retrieving data for a new user. (This can be addressed using either the $cacheFactory or $q APIS for this)
  • Repository values are recursive JSON objects - will need to use recursion down the sub-object tree of the repo value and enhanced styling to make for a clearer UI
  • Filters and sorting would be nice to have
  • Fancier styling, maybe?
